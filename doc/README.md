# WebCom 文档索引

WebCom 是一个轻量级、无框架依赖的前端组件库，提供语言工具函数、UI 组件以及数据驱动的自动化组件三大层次。

## 文档目录

| 文档 | 说明 |
|------|------|
| [项目概览](./overview.md) | 架构设计、代码规范、目录结构、构建方式 |
| [Lang 工具函数库](./lang.md) | 所有底层工具函数详细 API 说明 |
| [Widget UI 组件库](./widget.md) | 所有 UI 交互组件详细 API 说明 |
| [Auto 自动化组件](./auto.md) | 基于 data-* 属性的自动挂载组件 API 说明 |
